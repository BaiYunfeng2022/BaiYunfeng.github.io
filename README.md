# 个人网站项目

这是我的个人网站，使用GitHub Pages托管。

## 功能特点

- 响应式设计，适配各种设备
- 包含以下部分：
  - 首页介绍
  - 关于我（包含技能展示）
  - 作品集
  - 博客区域
  - 联系表单
- 平滑滚动导航
- 交互式表单

## 使用方法

1. 克隆此仓库到本地
2. 编辑`index.html`文件，将个人信息替换为您自己的信息
3. 根据需要修改`style.css`来自定义样式
4. 提交更改到GitHub仓库
5. 网站将自动部署到`https://github.com/BaiYunfeng2022/BaiYunfeng.github.io`

## 自定义

- 更新`index.html`中的内容
- 修改`style.css`中的颜色和样式
- 在GitHub仓库设置中启用自定义域名（可选）
- 添加更多页面（about.html, portfolio.html等）

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (原生)
- GitHub Pages (托管)
